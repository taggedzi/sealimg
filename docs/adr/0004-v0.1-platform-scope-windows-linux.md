# 0004 - v0.1 Platform Scope: Windows and Linux

- Status: Accepted
- Date: 2026-03-06

## Context
Sealimg needs a clear support target for initial implementation, packaging, and CI.

## Decision
Support Windows and Linux for v0.1.

## Consequences
- CI, packaging, and test fixtures must validate on both platforms.
- Path handling, filename behavior, and key storage conventions must be platform-aware.
- macOS support is deferred and should be treated as post-v0.1 unless explicitly reprioritized.

## Alternatives Considered
- Add macOS in v0.1: broader support but increased release complexity and test matrix size.
